みたいな。



左手でほぼキーボードが必要な動作をすべてこなせるように
BS、エンター、スペース、ホーム、エンド、シフトおしっぱ、タスク切り替えなどなど。
文字入力以外は右手マウス左手キーボードで出来るようにするのが目標。

多分世界で一番連打されててコレでもかというほど押しやすい位置にいるスペースキーを修飾キーにして使い倒すスクリプト。
ついでにその横に鎮座する絶好のポジションでありながら使い道に困る変換キーも同時に使い倒すスクリプト。
AHK搭載のコンビネーションキー（＆で書く奴）では一切縛っていないから、他のプログラムとも同居可能。
これならわざわざサスペンドしなくてもいいからプログラムごとにじゃまにならないはず。他のAhkファイルとも干渉しないはず。
おまけと言ってはなんだけどAHK純正のコンビネーションキーより高機能を目指している。

あとやりたいこと。動作アイコン変換とか細々としたとこ。

保存するときはUTF-8ですること。動作上支障はないが日本語が化けるので面倒。

とりあえずひな形はできたので手直ししたい人はどんどんどうぞ。著作権とか権利主張とか一切しませんし。
んで完成度高めてうｐしてくれると嬉しいです。


ファイル構成

OMKeybord　本体。
Componentsフォルダ
	keyconfigフォルダ
		各キーahkファイル。デフォルトのホットキーの動作を変えたいときはここで。
	Softwarebyフォルダ
		プログラム別ahkがカテゴリ別に入ってる
	AHKfileload.ahk　キーやプログラム別の呼び出しをまとめるファイル
	Oneshotmainfile.ahk　ワンショット本体。変換キー差し替えもここでやってるので気になったらここで変更。
	Templatetext.ahk　定型文用。オマケ
	UserScript.ahk　俺スクリプトを使いたいときはここに追加。オマケ



デフォルト設定

F1でサスペンド。

F12
ワンショットモデファイヤを動かすかどうかのスイッチが入ってる。

スクロールロック
フラグの状態を見れるキー。

無変換
こいつも押しやすいくせに使い道がない。
だからコピー切り取りペーストに割り当て。

1回でコピー
2連打でカット
押しっぱなしでペースト



スペース
ポンと１回押して離すとスペース発射。
押しっぱなしで何もしない。（間違えて押しちゃった時用）
押しっぱなし中各キーで他の動きをする

変換キー
ポンと１回押して離すと半角全角切り替え。純正の変換は使わんし、半角全角は使いまくるくせに遠いからこうした。
押しっぱなしで何もしない。（間違えて押しちゃった時用）
押しっぱなし中各キーで他の動きをする
要するにスペースキーと似たような動き。



以下コンビキー。



スペース＋変換キー
変換キー＋スペース
スペースを打ち出す。
連打用。

スペース＋wasd各キー
上下左右キー。ゲーム操作と一緒。

スペース＋qeキー
qがバックスペース。兼ブラウザバック。
eがブラウザ前進。

スペース＋12キー
home,end,pgup,pgdnの4キーをまとめている。

1でpageup。押しっぱなしでhome
2でpagedown。押しっぱなしでend

スペース＋34キー
タスク切り替え。alttabをフラグスイッチで組み込めるように設定。

スペース＋５
デスクトップ表示。

スペース＋tabキー
エンター。右だけにあるエンターは地味に使いにくいし。capslockもエンター化。

スペース＋半角全角
del割り当て。

スペース＋ｚキー
一つ前に戻る。ctrl+zの動き。

スペース＋ｘキー
一つ前に戻ったのをやめる。ctrl+yの動き。こっちのほうが直感的な気がする。

スペース＋ｙhキー
音量updown。どれくらいかを知るために音も出る。配置の理由はキーボードのど真ん中に居座ってるから。

スペース＋ｒ
リロード。F5キー。

スペース＋f
検索。ctrl+f。

スペース＋jkluio789m<>各キー
それぞれテンキーの123456789が入っている。
mは0。<には00。>にはドット。
人差し指のホームポジションから入力できるのは楽。

スペース＋ｖキー
エクスプローラ起動。あんまり使わないのでここに。

スペース＋ｂキー
タスクトレイにフォーカス移動。地味に使う。

スペース＋Nキー
左winキー。手のひらに隠れちゃうし。




変換キー＋;:]@
上下左右キー。右手版。Numpad上下左右を割り当てている。

変換キー＋p[
pがブラウザバック。BSではない。
[がブラウザ前進。

等など。
